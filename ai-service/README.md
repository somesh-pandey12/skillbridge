# SkillBridge — AI Career Portal

[![CI](https://github.com/yourname/skillbridge/actions/workflows/ci.yml/badge.svg)](...)

> Upload your resume. Get AI-powered skill gap analysis. Land your dream job.

## Features
- 🔐 Google OAuth authentication
- 📄 PDF resume parsing with Groq LLaMA 3.3
- 🎯 Skill gap analysis per company
- 📊 LeetCode topic recommendations
- 🔍 Semantic job matching via Pinecone

## Tech Stack
| Layer | Tech |
|-------|------|
| Frontend | React + Vite + Tailwind |
| Backend | Node.js + Express + MongoDB |
| AI Engine | Python FastAPI + Groq |
| Vector DB | Pinecone |
| Auth | Google OAuth 2.0 + JWT |
| Deploy | Vercel + Railway + Render |
